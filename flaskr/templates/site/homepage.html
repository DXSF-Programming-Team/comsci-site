Welcome to the DXSF CS Club!

On this website you can find information about the club, our recent puzzles and projects, and some coding-related resources. Check out the About page for more information about the club, and the Puzzles page for our recent puzzles and solutions.

If you are interested in joining the club, there is no formal signup necessary; just show up to a meeting and you're in! It is helpful if you contact us at 27mortry@dextersouthfield.org so we can add you to the mailing list. You do not need to know anything about coding--all we require is a desire to learn and dedication to the learning process.

Please note that this site is a work in progress, and we are in the process of adding new features and content. We apoligize for any inconvenience this may cause!

The code for this site is open source; we wrote it in Python using Flask, and we are hosting it on Render. The code is available on GitHub.
